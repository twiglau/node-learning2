# 框架设计: 考虑 4 大块内容

```lua
控制器          - - - >   处理请求

Services

Modules
管道            - - - >   核验请求的数据
Pipes

Filters

Guards

Interceptors
存储库          - - - >   处理在数据库中数据
Repositories
```
